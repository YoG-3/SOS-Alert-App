# SOS Alert App

An Android emergency alert application built using Kotlin and Jetpack Compose.

## Features
- One-tap SOS button
- Emergency contact management
- SMS alerts
- Email alerts
- Location sharing
- Alarm and vibration
- Custom alert messages
- Modern Jetpack Compose UI

## Technologies Used
- Kotlin
- Jetpack Compose
- Android Studio
